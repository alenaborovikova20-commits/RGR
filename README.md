# Cryptum — Криптографический инструмент

**Cryptum** — это консольное приложение для шифрования и дешифрования текста и файлов с использованием симметричных и асимметричных алгоритмов. Проект реализован на C++ с поддержкой динамических плагинов.

## Особенности

- Поддержка **6 алгоритмов** (5 симметричных + 1 асимметричный).
- Динамическая загрузка плагинов (`.so`).
- Шифрование/дешифрование **текста** и **файлов** (бинарных, текстовых, изображений).
- Генерация ключей и сохранение в файл.
- Интерактивное меню и поддержка аргументов командной строки.
- Безопасное затирание ключей в памяти (`secure_erase`).

## Поддерживаемые алгоритмы

| Алгоритм | Тип | Размер ключа | Размер блока |
|----------|-----|--------------|--------------|
| **RC5** | Симметричный | 16 байт | 8 байт |
| **XTEA** | Симметричный | 16 байт | 8 байт |
| **GOST 28147-89** | Симметричный | 32 байта | 8 байт |
| **Threefish-512** | Симметричный | 64 байта | 64 байта |
| **Blowfish** | Симметричный | 16 байт | 8 байт |
| **Goldwasser-Micali (GM)** | Асимметричный | 32 байта | 1 байт |
