# Cryptum — Криптографический инструмент

**Cryptum** — это консольное приложение для шифрования и дешифрования текста и файлов с использованием симметричных и асимметричных алгоритмов. Проект реализован на C++ с поддержкой динамических плагинов.

## Особенности

- Поддержка **6 алгоритмов** (5 симметричных + 1 асимметричный).
- Динамическая загрузка плагинов (`.so`).
- Шифрование/дешифрование **текста** и **файлов** (бинарных, текстовых, изображений).
- Генерация ключей и сохранение в файл.
- Интерактивное меню и поддержка аргументов командной строки.
- Безопасное затирание ключей в памяти (`secure_erase`).

## Поддерживаемые алгоритмы

| Алгоритм | Тип | Размер ключа | Размер блока |
|----------|-----|--------------|--------------|
| **RC5** | Симметричный | 16 байт | 8 байт |
| **XTEA** | Симметричный | 16 байт | 8 байт |
| **GOST 28147-89** | Симметричный | 32 байта | 8 байт |
| **Threefish-512** | Симметричный | 64 байта | 64 байта |
| **Blowfish** | Симметричный | 16 байт | 8 байт |
| **Goldwasser-Micali (GM)** | Асимметричный | 32 байта | 1 байт |

## Сборка

### Требования
- **C++17**
- **CMake 3.10+**
- **GCC/Clang** (Linux)
- **Git** (опционально)

### Инструкция

```bash
# Клонировать репозиторий
git clone https://github.com/alenaborovikova20-commits/RGR.git
cd RGR

# Собрать проект
mkdir build && cd build
cmake ..
make -j4

# Запустить
cd ..
./bin/cryptum
Использование
Интерактивное меню
bash
./bin/cryptum
После запуска откроется меню:

text
КРИПТОГРАФИЧЕСКИЙ ИНСТРУМЕНТ
1. Шифрование/дешифрование текста
2. Шифрование/дешифрование файла
3. Генератор ключей
4. Информация о формате ключей
0. Выход
Аргументы командной строки
bash
./bin/cryptum -a <алгоритм> -m <режим> -i <входной_файл> -o <выходной_файл> [-k <ключ>]
Примеры:

bash
# Шифрование с генерацией случайного ключа
./bin/cryptum -a rc5 -m encrypt -i input.txt -o encrypted.bin

# Шифрование с ключом из файла
./bin/cryptum -a rc5 -m encrypt -k key.bin -i input.txt -o encrypted.bin

# Расшифрование
./bin/cryptum -a rc5 -m decrypt -k key.bin -i encrypted.bin -o decrypted.txt

# Генерация ключа
./bin/cryptum -a rc5 -m generate-key -o key.bin

# Справка
./bin/cryptum -h
